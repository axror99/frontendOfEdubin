About us

Welcome to Edubin

EduBin is an online course website and it was founded on March 16, 2023. The founder is Bakhromaliyev Akhror This person is a java web developer and the website is developed by this person. The main goal of the website is to deliver high-quality and affordable online courses to students across Uzbekistan. The website is also engaged in online book sales. Users can get information on the website about where and when events of various content will be held, and can read the latest articles on various topics.
